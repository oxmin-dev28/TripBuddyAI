# TripBuddy AI — демо «Перезагрузка на 4 дня»

Репозиторий очищен от старых экранов и сервисов и сфокусирован на одном сценарии: 5-шаговый поток под быструю перезагрузку с готовыми вариантами А/Б (Лиссабон и Анталья) и CTA на билеты/сохранение плана. Цветовая система закреплена в четырёх оттенках: `#F5F8FC`, `#1E2A44`, `#141D2D`, `#0E141E`.

## Быстрый старт

1. Перейди в вложенную папку с приложением:
   ```bash
   cd TripBuddyAI
   ```
2. Установи зависимости (Expo + React Native):
   ```bash
   npm install
   ```
3. Запусти веб-версию демо:
   ```bash
   npm run web
   ```
   Если нужен Expo Go / эмулятор, используй `npx expo start` и выбери нужную платформу (клавиши `w`, `a`, `i`).

## Что осталось в коде

- `AppNavigator` ведёт сразу на `RebootFlowScreen`.
- UI-примитивы (`Button`, `Card`, `SelectableChip`, `ProgressBar`) используют обычные `StyleSheet`-стили на четырёх закреплённых цветах — без Babel-плагинов и tailwind.
- Лишние сервисы, экранные стеки, хранилище и мок-данные удалены, чтобы не мешали демо.

## Подсказки по сборке
- Если Expo попросит аккаунт, нажимай **Proceed anonymously**.
- При ошибках npm/registry попробуй `npm install --legacy-peer-deps`.
