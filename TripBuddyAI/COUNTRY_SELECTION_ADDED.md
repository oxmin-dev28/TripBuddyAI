# ✅ Добавлен выбор страны при создании маршрута

## 🎯 Что сделано:

### 1. **Новый экран SelectCountryScreen** ✨
- Показывает все доступные страны с флагами
- Популярные страны отображаются первыми
- Поиск по странам
- Показывает количество городов в каждой стране
- Красивые карточки с выбором

### 2. **Обновлён SelectCityScreen** 🔄
- Теперь принимает `countryId` из навигации
- Показывает только города выбранной страны
- Отображает название страны в заголовке
- Упрощённый интерфейс (без группировки по странам)

### 3. **Обновлена навигация** 🗺️
- Новый flow: **Home → SelectCountry → SelectCity → PlaceSelection → GeneratePlan**
- Плавные анимации переходов

---

## 📱 Новый Flow создания маршрута:

1. **Home** → Нажать "Создать маршрут 🚀"
2. **SelectCountry** → Выбрать страну (Франция, Италия, Испания...)
3. **SelectCity** → Выбрать город в выбранной стране
4. **PlaceSelection** → Выбрать места
5. **GeneratePlan** → ИИ генерирует маршрут
